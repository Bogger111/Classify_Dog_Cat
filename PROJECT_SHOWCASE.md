# 猫狗图像分类项目成果展示

## 项目简介

基于 PyTorch 实现猫狗图像二分类系统，自定义 ResNet18 风格残差网络，完成数据预处理、模型训练、验证评估、权重保存和结果可视化。项目使用 25,000 张猫狗图片进行训练与验证，最佳验证准确率达到 96.56%，并自动生成损失曲线、准确率曲线、混淆矩阵和样例预测图，具备进一步扩展为图片上传识别应用的基础。

## 实现内容

项目使用自定义 ResNet18 风格卷积神经网络作为核心模型。模型包含残差块、批归一化、ReLU 激活、自适应平均池化和 Dropout，能够在较深网络中保持稳定训练，并提升图像特征提取能力。

训练流程采用 Adam 优化器、CrossEntropyLoss 损失函数和 StepLR 学习率调度策略。训练过程中会自动记录训练集与验证集的损失和准确率，并保存最佳模型权重与最后一轮模型权重，便于后续复现、评估和部署。

数据处理部分支持直接读取图片文件，也支持使用预处理后的 Tensor 文件加速训练。训练时加入随机裁剪、水平翻转和归一化等图像增强方法，提高模型对不同图片姿态、构图和光照变化的适应能力。

## 数据与训练设置

- 数据规模：训练目录共 25,000 张图片，其中猫 12,500 张、狗 12,500 张。
- 数据划分：默认按 8:2 划分训练集和验证集，约 20,000 张用于训练，5,000 张用于验证。
- 训练轮数：50 轮。
- Batch Size：16。
- 优化器：Adam，学习率 0.001，权重衰减 0.0001。
- 损失函数：CrossEntropyLoss。
- 学习率调度：StepLR，每 20 轮衰减一次，衰减系数 0.8。
- 模型保存：保存 `best` 最优权重和 `last` 最后一轮权重。

## 项目成果

模型在验证集上取得了较好的分类效果，最佳验证准确率达到 **96.56%**。最后一轮训练后，训练准确率为 **96.56%**，验证准确率为 **95.20%**，说明模型已经能够较稳定地区分猫狗图片，并具备较强的泛化能力。

训练过程中的损失曲线和准确率曲线显示，模型在训练过程中逐步收敛，准确率持续提升。项目还生成了混淆矩阵和样例预测图，可以直观展示模型在验证集上的分类表现，以及模型对具体图片的预测结果。

主要输出文件包括：

- `outputs/loss_curve.png`：训练集与验证集损失变化曲线。
- `outputs/accuracy_curve.png`：训练集与验证集准确率变化曲线。
- `outputs/confusion_matrix.png`：验证集分类混淆矩阵。
- `outputs/sample_predictions.png`：样例图片预测结果展示。
- `checkpoints/Adam_lr0.001_CrossEntropyLoss_StepLR_best.pth`：最佳模型权重。
- `checkpoints/Adam_lr0.001_CrossEntropyLoss_StepLR_last.pth`：最后一轮模型权重。

## 技术亮点

1. 完整的深度学习训练流程  
   项目覆盖数据读取、图像增强、模型训练、验证评估、权重保存和结果可视化，结构完整，便于复现和扩展。

2. 自定义残差网络结构  
   通过实现 ResNet 风格的残差块，缓解深层网络训练中的梯度传播问题，使模型能够学习更丰富的图像特征。

3. 自动化结果可视化  
   训练结束后自动生成损失曲线、准确率曲线、混淆矩阵和样例预测图，使模型效果不只停留在数字指标上，也能通过图像直观呈现。

4. 支持训练加速与断点保存  
   项目支持 AMP 混合精度训练、CUDA 加速、预处理 Tensor 数据读取，以及最佳权重和最后权重保存，提升训练效率和实验管理能力。

5. 具备应用化扩展基础  
   项目中包含 Streamlit 应用入口，可继续扩展为上传图片并实时判断猫狗类别的可视化界面。
